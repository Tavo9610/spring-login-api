# Spring Auth API

API REST desarrollada con Spring Boot.

## Tecnologías que use

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Swagger/OpenAPI

## Funcionalidades 

- CRUD de usuarios
- Login básico con Spring Security
- Password hashing con BCrypt
- Manejo global de excepciones
- DTOs
- Arquitectura por capas

## Endpoints que expone

- POST /users
- GET /users
- GET /users/{id}
- PUT /users/{id}
- DELETE /users/{id}
